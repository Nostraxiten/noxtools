# NOSTRAXITEN DOMINION v2.2
## Framework de Auditoría y OSINT para Entornos Móviles

NOSTRAXITEN DOMINION es una suite profesional diseñada para la recolección de inteligencia de fuentes abiertas (OSINT) y pruebas de penetración. El ecosistema se integra mediante un núcleo centralizado que permite la ejecución de 11 módulos especializados.

### Módulos Integrados
* **Cat's Eye**: Motor de búsqueda de perfiles en plataformas sociales.
* **Adox**: Sistema de recolección de datos y reportes de información personal.
* **NoxCheck**: Analizador de disponibilidad y estados en protocolos de mensajería.
* **NoxPhone**: Algoritmo de generación y validación de numeración internacional.
* **AutoPC**: Módulo de auditoría orientado a sistemas operativos de escritorio.
* **Nopsi**: Servicio de geolocalización y análisis de procedencia de direcciones IP.
* **Nox MX**: Auditoría de configuración de servidores de correo y registros MX.
* **NoxSee**: Extracción de metadatos y análisis de cuentas de comunicación directa.
* **NoxMP**: Capturador de direcciones IP mediante vectores de redirección.
* **NoxWS**: Analizador de tráfico y origen de URLs de mensajería.
* **NoxIF**: Interfaz avanzada para la monitorización de tráfico de red.

### Protocolo de Instalación
Para el correcto funcionamiento, se requiere la instalación previa de Git, Python 3.x, PHP y Curl.

```bash
git clone https://github.com/nostraxiten/noxtools.git
cd noxtools
chmod +x *.sh
bash noxtool.sh
```
